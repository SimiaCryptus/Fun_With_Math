Mandelbrot Set

An interactive fractal explorer. Scroll to zoom, drag to pan.

 

Controls

About

The Mandelbrot set contains every complex number c for which the sequence z₀=0, zₙ₊₁=zₙ²+c stays bounded. Points inside are coloured black; outside points are coloured by escape speed.

Navigation

  • Scroll – zoom in / out
  • Drag – pan around
  • Double-click – zoom to point
